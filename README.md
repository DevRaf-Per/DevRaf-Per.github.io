# ALL ABOUT THIS SITE

## Why?

This site was created as a sandbox, to develop, test, and practice coding (free-of-charge) for the static portions of my “real” website. I wanted to learn how to create my own fully functioning site, with all the capabilities of a B2C site; without the help of the so-called “free” website builders you see advertised so ubiquitously (They all come at a price! Some site builders clearly state what they expect in return, others are much worse, with the true cost nefariously hidden deep within the free code they use).

This meant I had to be up-to-date with the latest in HTML, CSS, and JS at the bare minimum, for a static site **such as this website**. And, to achieve my ultimate goal, of a dynamic site as almost all B2C sites are[^1]; it also meant jQuery, AJAX, PHP, ASP, XML, SQL. Not to mention all the “auxiliary” tools needed for a “pretty” site, and any languages needed to create an application that may accompany the site…

And I wanted to do all this, without spending a single dime. Daunting, right? ...But doable! And, since I’ve been pursuing this goal since I was literally a child 😊, and over the past decade or two I’ve managed to learn almost all I need (if they would just stop updating and evolving!!!)

[^1]: Since dynamic websites require a server and storage, I am using the services provided by the big 3 web-service providers out there. And to maintain the free-of-charge goal, and ensure security of the site, I’m not sharing links to those sites yet. They are **coming soon**...
